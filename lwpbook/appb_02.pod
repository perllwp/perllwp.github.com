=head1 B.2. 200s: Successful

If an LWP request gets any of these codes, C<$response-E<gt>is_success>
will be true. For all other status codes, C<$response-E<gt>is_success>
will be false.

 200 OK
 201 Created
 202 Accepted
 203 Non-Authoritative Information
 204 No Content
 205 Reset Content
 206 Partial Content
