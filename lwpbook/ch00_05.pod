=head1 0.5. Conventions Used in This Book

The following typographic conventions are used in this book:

=over

=item I<I<Italic>>

Used for file and directory names, email addresses, and URLs, as well
as for new terms where they are defined.

=item C<Constant width>

Used for code listings and for keywords, variables, function names,
command options, parameters, and bits of HTML source where they appear
in the text.

=item I<C<B<Constant width bold>>>

Used to highlight key fragments of larger code examples, or to show the
output of a piece of code.

=item I<C<I<C<Constant width italic>>>>

Used as a general placeholder to indicate terms that should be replaced
by actual values in your own programs.

=back

=cut
