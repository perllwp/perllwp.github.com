=head1 B.3. 300s: Redirection

If an LWP request gets any of these codes,
C<$response-E<gt>is_redirect> will be true. For all other status codes,
C<$response-E<gt>is_redirect> will be false.

 300 Multiple Choices
 301 Moved Permanently
 302 Found
 303 See Other
 304 Not Modified
 305 Use Proxy
 307 Temporary Redirect

=cut
